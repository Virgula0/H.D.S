Welcome to H.D.S

Hashcat Distributed Service

Project Description

H.D.S (hashcat-distributed-service) is a university project entirely written in Go that can find practical applications for distributing handshakes cracking tasks on multiple hashcat clients. It is mainly designed for cracking WPA2 handshakes capturing and converting `PCAP` files using bettercap plus a daemon designed to send such handshakes to the server in a simple, lightweight and handy way.

The daemon checks for correct WPA captures by examining EAPOL messages, and if a full 4-way handshake is found, it will be sent to the user on the front-end. Clients wait for tasks submitted by users from the dashboard and proceed to use hashcat whenever requested.

The project will be expanded to support all handshakes cracking tasks, not only WPAs.
